# Rice Genotype–Trait Multi-Analysis Pipeline

This repository provides a complete statistical and machine learning
pipeline for analyzing genotype–trait relationships in rice breeding experiments.

The workflow integrates:

• BLUP-based heritability estimation  
• Graphical Lasso trait networks  
• Multiple linear regression yield drivers  
• PLS regression + VIP scoring  
• Random forest & SHAP feature importance  
• MGIDI multi-trait selection index  
• Path coefficient analysis  
• Trait correlation networks  
• High-resolution publication-ready figures

The pipeline was designed for use with RCBD field experiments.
